# 📊 Meta Ad Performance Dashboard | Power BI

## 📌 Project Overview

The **Meta Ad Performance Dashboard** is an interactive Business Intelligence solution built using **Power BI** to analyze and monitor the performance of Meta (Facebook & Instagram) advertising campaigns.

The dashboard transforms raw marketing data into meaningful insights, enabling users to evaluate campaign effectiveness, audience engagement, conversions, and overall marketing performance through interactive visualizations.

---

## 🎯 Project Objectives

- Analyze advertising campaign performance.
- Track user engagement and conversions.
- Monitor important marketing KPIs.
- Identify audience behavior and demographic trends.
- Support data-driven marketing decisions.

# 📂 Dataset

The dashboard is developed using three interconnected datasets.

### 1️⃣ Campaigns Dataset
Contains campaign-related information such as:

- Campaign ID
- Campaign Name
- Platform
- Campaign Budget
- Campaign Duration

---

### 2️⃣ Users Dataset

Contains customer demographic information including:

- User ID
- Age
- Gender
- Country
- Interests

---

### 3️⃣ Ad Events Dataset

Contains advertising event details including:

- Impressions
- Clicks
- Likes
- Shares
- Purchases
- Event Timestamp

---

# 🧹 Data Preparation

Before building the dashboard, the data was cleaned and transformed using **Power Query**.

The process included:

- Removing duplicate records
- Handling missing values
- Correcting data types
- Standardizing column names
- Creating relationships
- Preparing data for analysis

---

# 🏗️ Data Model

The project follows a relational data model connecting:

- Campaigns
- Users
- Ad Events

This enables efficient filtering and accurate KPI calculations.
